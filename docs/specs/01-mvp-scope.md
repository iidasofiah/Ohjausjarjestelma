# MVP-scope

## MVP sisältää
1) Yksi case: IAQ-ohjattu ilmanvaihto (simuloidulla datalla)
2) Sisääntulo: CO2, lämpötila, kosteus (simuloitu)
3) Ulostulo: ilmanvaihdon teho 0-100% (simuloitu)
4) Sääntömoottori: kynnysarvot + hystereesi (ajetaan esim. 10s välein)
5) UI: Dashboard (tila + trendi), sääntöjen muokkaus, tapahtumaloki
6) Käyttäjätasot: Admin / Operator / Viewer
7) Perustason tietoturva: kirjautuminen + roolipohjainen pääsy

## MVP ei sisällä (vielä)
- Oikeat laiteintegraatiot (vain simulaatio)
- Edistynyt optimointi / ML
