# Visio ja tavoitteet

## Tavoitteet
- Säätö useiden sisääntulojen ja ulostulojen välille (muokattava/mukautettava)
- Helppokäyttöinen selainpohjainen UI (mobiili + desktop)
- Modulaarinen rakenne eri käyttökohteisiin:
  - IAQ -> ilmanvaihto
  - Pumppaus -> pinnankorkeus + nousunopeus
  - Lämmitys jne.
- Toimii taustalla ilman aktiivista selainikkunaa (engine/worker)
- Tilatiedot, status, toimintahistoria/audit

## Yleiset reunaehdot
- Generoidaan tarvittava laitedata kehitykseen
- Tietokanta tarvitaan
- Tietoturvallisuus ja käyttäjätasot kaikissa caseissa
- Vaiheittainen speksaus ja dokumentointi
- Välipalaverit noin 2 viikon välein
